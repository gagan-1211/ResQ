Data Structures Being Tested:

Emergency Alerts

Resources

{

    name: string,
    quantity: number,
    location: {
        latitude: number,
        longitude: number,
        address: string
    },

}

Volunteers

{
    personalInfo: {
        fullName: string,
        email: string,
        phone: string
    },
    skills: string[],
    availability: {
        status: 'ACTIVE' | 'INACTIVE',

}

NGOs

{

        name: string,
        registrationNumber: string,
        contact: string
    },
    expertise: string[],

Relief Camps
